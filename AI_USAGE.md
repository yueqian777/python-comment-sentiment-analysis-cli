- 工具名称：OpenAI Codex 桌面应用

- 可确认模型标识：GPT-5 Codex 编程代理

客户端版本通过 Windows `Get-AppxPackage -Name OpenAI.Codex` 查询。直接运行 `codex --version` 时，WindowsApps 安装目录返回“Access is denied”，因此不把不可执行命令的结果伪装成 CLI 版本。

## 使用范围

本项目中 Codex 用于：

- 分析课程项目要求；
- 辅助重构 Python 代码并设计 pytest 测试，包括预处理一致性、单字匹配保护和评估复现信息；
- 辅助整理训练数据和固定留出测试集 v1 的评论及标签草稿，使用者需要逐条复核；
- 在相关函数附近添加简短 AI 辅助注释，完整披露仍以本文为准；
- 执行本地测试、训练、交叉验证和输出文件检查。

使用者仍需检查数据标签，理解并能够解释文本清洗、词典匹配、Pipeline、交叉验证和留出评估等核心逻辑。课程若禁止 AI 直接生成核心逻辑，应按教师要求进一步处理。本文件只做真实披露，不代表自动满足全部课程规定，也不声称使用了其他 AI 服务。
