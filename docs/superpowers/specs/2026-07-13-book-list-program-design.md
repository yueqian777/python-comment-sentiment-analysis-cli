# 书单兴趣清单程序设计

## 目标

编写一个适合 Python 基础课程作业的命令行“我的书单”程序。程序直接使用列表保存数据，结构简单、便于阅读和课堂讲解。

## 数据结构

使用一个列表保存全部书籍，列表中的每一项是包含以下字段的字典：

- 书名
- 作者
- 类型
- 状态

程序启动时预置《活着》《三体》《百年孤独》三本示例书。数据仅在程序运行期间有效，退出后不保存。

## 功能

主菜单循环提供四个选项：

1. 添加书籍：依次输入书名、作者、类型和阅读状态，并添加到列表。
2. 查看全部：按编号显示列表中的所有书籍及其四个字段。
3. 关键词查询：在书名、作者、类型和状态中进行包含匹配，并显示所有匹配项。
4. 退出程序：结束菜单循环并显示退出提示。

## 输入与异常处理

- 菜单输入不是 1 至 4 时，显示提示并重新显示菜单。
- 添加书籍时，书名或作者为空则提示重新输入。
- 阅读状态由使用者直接输入，示例使用“未读”“在读”“已读”。
- 查询不到结果时显示“没有找到相关书籍”。
- 关键词为空时提示重新输入，避免无意义地匹配全部记录。

## 代码结构

程序保存在一个独立 Python 文件中。添加、显示、查询和主菜单分别使用函数实现，既保持初学者可读性，也方便自动化测试。

## 验证

自动化测试覆盖添加书籍、展示全部书籍、关键词查询和无结果查询。最后通过模拟菜单输入完成一次交互冒烟测试，确认程序可以正常退出。
